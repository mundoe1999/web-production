Facilities in the Computer Science Department

Instructional Equipment and Facilies

All computer science students can use any of the general-purpose labs throughout Hunter College. In addition, computer science majors and students enrolled in any of CSCI 127, 132, 135, 136, or 232, or BIOL 375 or 425 can obtain an account on the Computer Science Department Network. Having an account on the network entitles the student to 24/7 access to "Lab 1000-G", as well as remote access through the network portal.

Learn more about the Lab and about the rules and regulations which govern its use HERE.

1000-G Lab

The 1000-G lab is located on the tenth floor of the North Building. It consists of 30 Dell Precision T3500n computers with the following specs:

  • 2-GB RAM
  • 256-MB ATI FireMV 2260 graphics cards
  • 2.4-GHz Dual Core Processors
  • 19-inch Ultrasharp 1908FP monitors
  • Runs RHEL 5.5 on a gigabit network

Servers

There is a Dell PowerEdge T610 file server running RHEL 5.5 with the following specs:

  • 12-GB RAM
  • two quad processor Xenon 2.4 GHz processors
  • eight 600-GB 10k RPM SCSI hard drives

The network portal is a Dell PowerEdge dual-processor that runs at 1-GB ram.
For the MySQL servers, it runs on Dell R200 with:

  • 2.8 GHz Xenon processor
  • 8-GB RAM
  • Runs RHEL5 with two 146-GB SCSI hard drives
